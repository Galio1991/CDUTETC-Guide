# 不要温和地走进那个厕所

这个版块不讲怎么复习、怎么投简历、怎么面试。

这个版块讲的是：**作为一个三本学生，在升学就业这条路上，你应该怎么想，怎么做。**

## 写在前面

"不要温和地走进那个厕所"——化用叶芝的诗。

三本学生的处境确实不容易，但不容易不等于没机会。你需要做的是：**看清现实，做出选择，然后执行。**

不要因为"我是三本"就放弃，也不要因为"我是三本"就盲目乐观。你需要的是：**清醒的认知 + 务实的行动。**
