# 三本学历的"反脆弱"策略

## 先接受一个事实

三本学历在就业市场上确实处于劣势。这是事实，不用美化。

但劣势不是死局。

## 学历的真正作用

学历是什么？是一块**敲门砖**。

- 985/211 的学生拿着金砖敲门，门很容易开
- 三本学生拿着木砖敲门，门不太容易开

但敲门砖只是第一步。进了门之后，**能力才是通行证**。

而且，门不止一扇。

## 三本学生的三个策略

### 策略一：曲线救国

**核心思路**：不去挤最热门的赛道，找竞争更小但前景不错的方向。

**具体做法**：

- 不盲目追求大厂，先进中小公司积累经验
- 不盲目追求热门行业，找有增长潜力的细分领域
- 不盲目追求一线城市，看看新一线和二线的机会

**例子**：

- 大厂的测试岗位竞争激烈，但中厂的测试岗位可能更容易进
- 互联网行业很卷，但传统行业的数字化转型需要大量人才
- 一线城市的竞争激烈，但成都、重庆、西安的机会也不少

### 策略二：能力证明

**核心思路**：用实际成果替代学历背书。

**具体做法**：

- 做项目，放到 GitHub 上
- 考证书，证明你的专业能力
- 参加比赛，拿到奖项
- 写博客，展示你的思考能力

**例子**：

- 一个有 1000 star GitHub 项目的三本学生，比一个只有学历的 985 学生更有竞争力
- 一个有 AWS 认证的三本学生，比一个没有认证的 211 学生更有竞争力
- 一个有省级竞赛奖项的三本学生，比一个没有奖项的一本学生更有竞争力

### 策略三：降维打击

**核心思路**：去竞争更小的池子里当大鱼。

**具体做法**：

- 去小公司，你能接触到更多核心业务
- 去传统行业，你的技术背景是稀缺资源
- 去创业公司，你的成长空间更大

**例子**：

- 在大厂你可能只是拧螺丝，在小公司你可能负责整个项目
- 在互联网行业你是普通一员，在传统行业你可能是技术骨干
- 在成熟公司你按部就班，在创业公司你可能快速成长

## 一个重要的认知

三本学历的劣势是**短期**的，能力的优势是**长期**的。

- 第一份工作，学历很重要
- 第三份工作，能力很重要
- 第五份工作，经验和资源很重要

你需要做的是：**在短期内弥补学历的劣势，在长期内建立能力的优势。**

## 具体行动清单

### 大一

- 搞清楚本专业的就业方向
- 开始学习一门实用技能
- 保持绩点在中上水平

### 大二

- 找到第一份实习（哪怕是小公司）
- 开始做项目，放到 GitHub 上
- 考一个相关的证书

### 大三

- 找到一份有质量的实习
- 确定毕业方向（考研/就业/考公）
- 开始准备（考研复习/秋招/考公）

### 大四

- 执行计划
- 总结经验
- 为下一步做准备
