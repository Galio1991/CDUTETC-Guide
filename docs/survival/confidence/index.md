# 欢迎来到三本

> 无论有多少人以怎样的方式祝福，每年有99.93%的人考不上清北，98.5%的人考不上985，96%的人考不上211。我们的教育资源是如此匮乏，以至于即使奇迹降临，一个人考上的后果不过是另一个人落榜。
>
> 因此，如果你有足够的幸运，足够的努力，足够的聪明考上顶尖大学，我祝福你在经过漫长的求学生涯后，仍然保持一定理想主义；我祝福你谋求改变这种匮乏，而不是以人上人自居；我祝福你仍然对最朴素的公义怀有敬畏，而非只谋求自己的私利；我祝福你发现即使所有的弊病都无法拯救，仍然愿意爱惜羽毛，而非同流合污。

## 写在前面

在三本里，你会发现自己的选择比你想象的更少，也比你承受的更多。课程表上写满了基础课、选修课和一堆无人问津的实验，但真正能改变你的，不在课本，也不在成绩单，而在你是否愿意去主动寻找属于自己的路径。你可能会羡慕985、211的同学拥有更丰富的资源，但那只是表象。机会永远不会凭学历落在你面前，它只会偏爱那些知道自己想要什么、并愿意付出实际行动的人。

你会遇到焦虑和困惑——焦虑于不知道该如何利用时间，困惑于选择哪些事情值得投入。在这样的环境里，最容易陷入的陷阱是等待和比较。你看着别人取得成就，而自己却停在原地，仿佛世界在提醒你，你比别人慢了一步。然而，这种焦虑并非完全来自学校，而是来自你对“成功路径”的误解。成功并非标准化的公式，也不在于平台本身，而在于你能否在有限的条件下，创造出属于自己的优势。

因此，欢迎来到三本。这不仅是一所学校的名称，更是一种环境，一种考验，一种逼迫你去面对真实自己的条件。你将在这里学会如何在极其有限的资源里最大化自己的成长，学会如何在不完美的环境下保持理想。

你的大学，不会因为三本而减少价值；你的努力，不会因为环境而失效。真正的考验，不是起点，而是你愿不愿意面对现实，并在现实之上，建造属于自己的未来。

这不是鸡汤，这只是编者对读者们的一点期望与祝福。
